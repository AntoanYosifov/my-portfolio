Core Program

Fundamentals

Fundamentals

Working with basic syntax, conditional statements and loops, data types and variables. Using Git and GitHub; HTTP basics; arrays, methods, lists, and text processing. Introduction to objects and classes, associative arrays, bitwise operations, regular expressions, and structured problem solving. Basics of HTML & CSS, database fundamentals, QA introduction, and a basic web project.

Java Advanced and OOP

Java Advanced and OOP

Working with linear data structures, multidimensional arrays, and associative collections. Streams, files, and directories; functional programming; defining classes; generics; iterators and comparators. Object-oriented programming: abstraction, encapsulation, inheritance, interfaces, and polymorphism; applying SOLID principles. Reflection and annotations, exceptions and error handling, unit testing, test-driven development, and design patterns.

Databases and ORM

Databases and ORM

Relational database fundamentals with MySQL: data definition and datatypes, CRUD operations, built-in functions, aggregation, table relations, subqueries and JOINs, programmability, and transactions. ORM and persistence: Hibernate introduction, Code First mapping, Spring Data fundamentals, advanced querying, and object mapping; JSON and XML processing for data exchange.

Web Development

Web Development

HTTP protocol and web fundamentals; Spring Core and Spring MVC; Spring Boot + MVC; server-side rendering with Thymeleaf; state management; events, scheduling tasks, and caching. Building REST APIs with Spring; Spring Security; microservice architecture; exception handling; unit testing, integration testing, and API testing; aspect-oriented programming; introduction to cloud and messaging with Spring for Apache Kafka.

Specialized Courses

HTML & CSS

HTML & CSS

Building responsive web pages with semantic HTML and modern CSS. Working with the box model, typography, positioning, and layout techniques (Flexbox). Styling forms and creating clean, consistent UI with a focus on responsive design and cross-device presentation.

JS Front-End

JS Front-End

JavaScript fundamentals for the browser: arrays and strings, functions, objects and classes. DOM manipulation and event handling for interactive UI. Working with HTTP and REST APIs, asynchronous programming, and client-side logic for dynamic, data-driven front-end applications.

Git

Git

Working with Git fundamentals and efficient branching workflows. Applying effective branch management, resolving merge conflicts, and maintaining clean history. Managing work-in-progress and restoring changes (e.g., stashing/reset/revert), plus automation and best practices with Git UI tools.

Angular

Angular

Building single-page applications with Angular and TypeScript. Working with components and dependency injection, services, and RxJS basics. Structuring apps with modules, routing, and signals; building forms and validation. Using pipes and interceptors, subjects, and applying state management patterns.
